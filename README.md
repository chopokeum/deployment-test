# deployment-test

## Install SDK

1. Open Package Manager
2. Click on the + icon on the top left corner of the "Package Manager" screen
3. Click on "Install package from git url..."
4. Paste: https://github.com/chopokeum/deployment-test.git?path=sdk

## Install Restricted SDK

1. Open Package Manager
2. Click on the + icon on the top left corner of the "Package Manager" screen
3. Click on "Install package from git url..."
4. Paste: https://github.com/chopokeum/deployment-test.git?path=sdk-restricted
